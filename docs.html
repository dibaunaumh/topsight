Topsight.js - Documentation

Topsight is a term coined by the computer scientist David Gelernter, to denote the ability to see the big picture of a complex system.
Topsight.js is an engine for constructing Zoomable UI, that enables seeing the big picture while still being able to zoom into the small details, just like in online maps.
An example for Zoomable UI is the presentation tool Prezi.

The framework arrives with a set of basic artifacts (widgets), named after physical artifacts for intuitiveness:

  • paper: for text input
  • label: a text label
  • wheel: a circle
  • pipe: a connector line

Artifacts are constructed as follows:

  • artifacts are always drawn in logical view: 0,0 - 100,100
  • artifacts are constructed in the coordinates system of their container. the top container also has 0,0-100,100 coordinates.
  • a context reference allows selecting other artifacts in the scene, using their name

For events handling, it uses the common JQuery syntax with selectors & event listeners.
