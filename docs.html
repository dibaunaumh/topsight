Topsight.js - Documentation

Topsight is a term coined by the famous Yale professor David Gelernter, to denote the ability to see the big picture of a complex system. Topsight.js is an engine for constructing Zoomable UI, that enables seeing the big picture while still being able to zoom into the small details, just like in online maps. A famous example for Zoomable UI is the presentation tool Prezi.

  • artifact are always drawn in logical view: 0,0 - 100,100
  • artifacts are constructed in the coordinates system of their container. the top container also has 0,0-100,100 coordinates.
  • a context reference allows selecting other artifacts in the scene, using their name
  • events are handled seperately, using jquery selectors syntax
  • framework arrives with a set of defined artifacts
  • framework arrives with zooming capability
