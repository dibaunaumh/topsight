Topsight.js - Documentation

  • artifact are always drawn in logical view: 0,0 - 100,100
  • artifacts are constructed in the coordinates system of their container. the top container also has 0,0-100,100 coordinates.
  • a context reference allows selecting other artifacts in the scene, using their name
  • events are handled seperately, using jquery selectors syntax
  • framework arrives with a set of defined artifacts
